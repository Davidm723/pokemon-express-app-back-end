# Pokemon Party and Box Manager

This project was inspired by my currect hyperfixation with Pokemon. I wanted a place where you can view and manage all your pokemon without have to trek back to the in-game PC to check it out.

[Pokemon Party and Box Manager](https://pokemon-react-app-front-end.onrender.com/)

## Technologies Used

- React
- React-Router
- Express
- JSX
- CSS
- Flexbox
- Dotenv
- Mongoose
- MongoDB
- BCrypt
- PokeApi
- CORS
- JWT

## Next Steps

I'd really like to be able to more so customize the interface. Allow users to add nicknames or change the sprite if they have a shiny pokemon.
